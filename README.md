
## 🤖 AI/ML INTERNSHIP PROJECTS
**📖 OVERVIEW**<br>
This repository contains a collection of Artificial Intelligence and Machine Learning projects completed during my AI/ML internship. The projects demonstrate practical implementation of data analysis, regression, classification, time-series forecasting, and chatbot development using Large Language Models.

Each project follows a structured workflow including problem definition, dataset preprocessing, exploratory data analysis, model training, evaluation, and result interpretation. <br>


🚀 PROJECTS INCLUDED<br> <br>
**1️⃣ Iris Dataset Exploration & Visualization**<br>
Objective: Explore feature relationships and data distributions within the Iris dataset.

Techniques Used: Data inspection (Pandas), Summary statistics, Scatter plots, Histograms, and Box plots.

Outcome: Identified clear feature patterns distinguishing different species. <br>


**2️⃣ Stock Price Prediction (Short-Term Forecasting)** <br>
Objective: Predict the next day’s closing stock price using historical market data.

Dataset: Yahoo Finance (via yfinance API).

Models Used: Linear Regression, Random Forest.

Outcome: Random Forest provided improved prediction accuracy over linear regression. <br>


**3️⃣ Heart Disease Prediction** <br>
Objective: Predict whether a patient is at risk of heart disease based on medical attributes.

Dataset: UCI Heart Disease Dataset.

Models Used: Logistic Regression, Decision Tree.

Evaluation: Accuracy, Confusion Matrix, ROC-AUC Score.

Outcome: Identified important medical features influencing disease prediction. <br>


**4️⃣ General Health Query Chatbot** <br>
Objective: Develop a chatbot capable of answering general health-related queries using an LLM.

Tools Used: OpenAI API / Hugging Face Models, Prompt Engineering, Safety Filtering.

Outcome: Built a conversational assistant designed to provide clear and safe health information. <br>


**5️⃣ Mental Health Support Chatbot (Fine-Tuned Model)** <br>
Objective: Create an empathetic chatbot providing supportive responses for emotional wellness.

Base Models: DistilGPT2 / GPT-Neo / Mistral.

Dataset: EmpatheticDialogues (Hugging Face Transformers).

Outcome: Fine-tuned a model to generate emotionally supportive and empathetic responses. <br>

**6️⃣ House Price Prediction** <br>
Objective: Predict property prices using real estate features.

Dataset: Kaggle House Price Dataset.

Models Used: Linear Regression, Gradient Boosting.

Evaluation: Mean Absolute Error (MAE), Root Mean Squared Error (RMSE).

Outcome: Developed a regression pipeline for accurate house price estimation.
